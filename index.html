Important Concepts Learnt in Lesson 1

How the Web Works
The major pieces of the web include a browser, the internet, a server and files. When a person goes on the internet, their computer sends an HTTP message to the server. The server finds theHTML file and sends it back to the person where web browser displays it.

HTML
HTML stands for Hypertext Markup Language. HTML documents contain text, markup, references and links to other pages.Here is a video explaining this.

Tags
Tags are used to enhance the way the text appears, and to attach links etc. Tags can be for adding images, making words bold or italic, making page breaks, paragraphs etc. A tag is a formatting tool as well as a tool to insert videos, links and images.

Inline and Block elements
Inline elements are tags that change the way something appears within the text contained in a block. Block elements are those that contain a block of content
